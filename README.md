# InventarioTienda-NetCore-React
Aplicación full stack para gestión de inventario de tienda. Backend en ASP.NET Core Web API con Entity Framework Core y SQLite. Frontend en React con CRUD completo, validaciones y feedback UX. Incluye integración opcional con Google Gemini API para generar descripciones de productos.

## Tecnologías
- Backend: ASP.NET Core Web API
- Frontend: React
- Base de datos: SQLite
- ORM: Entity Framework Core

## Estructura del proyecto
- Backend/: API REST desarrollada en .NET Core
- Frontend/: Aplicación React
- README.md: Instrucciones del proyecto

## Requisitos
- .NET SDK 7 o superior
- Node.js (para el frontend)
- Git
